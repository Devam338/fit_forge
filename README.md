# FitForge

An easy-to-use fitness/diet tracker

<img width="1680" height="962" alt="Screenshot 2026-04-14 at 9 14 12 PM" src="https://github.com/user-attachments/assets/9be51589-8ae6-4a2e-8904-dc762f167c72" />

## Included pages
- Dashboard
- Workouts
- Diet Tracker
- GPS Activity
- Progress Tracking
- AI Coach
- Achievements
- Notifications
- Settings

This keeps the app fully demoable and easy to deploy on Vercel without setting up a database.

## Tech stack
- Next.js 14
- TypeScript
- Recharts
- lucide-react

## Run locally
```bash
npm install
npm run dev
```
