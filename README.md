# FitForge

An easy-to-use fitness tracker

## Included pages
- Dashboard
- Workouts
- Diet Tracker
- GPS Activity
- Progress Tracking
- AI Coach
- Achievements
- Notifications
- Settings

This keeps the app fully demoable and easy to deploy on Vercel without setting up a database.

## Tech stack
- Next.js 14
- TypeScript
- Recharts
- lucide-react

## Run locally
```bash
npm install
npm run dev
```
